================================================================================
1. PROJECT OVERVIEW
================================================================================

Goal: Identify functional sub-families within protein families by combining
protein language model (PLM) embeddings with phylogenetic comparative methods
(PCMs). The key insight is to treat PLM embeddings as continuous multivariate
phenotypes evolving on a phylogenetic tree under a Brownian Motion (BM) model
with Matrix Normal distribution.

Method name: KAVERET
  (Kronecker Analysis of Variance in Embedding Representation Evolutionary Trees)

Two-stage statistical procedure:
  Stage 1: Recursive Phylogenetic ANOVA (RRPP) detects mean-shift sub-families
  Stage 2: LRT for V_A = V_B (coordinate covariance equality) with
           Westfall-Young max-statistic bootstrap for multiple testing correction

================================================================================
3. PIPELINE FLOW (full_pipeline_mean_shift.py --operation full)
================================================================================

PHASE 1: PREPROCESS
  - Stockholm -> FASTA conversion
  - MSA statistics (n_seqs, avg_length, sequence similarity, etc.)
  - InterPro/Pfam metadata fetch (description, protein category flags)

PHASE 2: FIND_BEST_SPLIT (evaluate_top_splits)

  Phase 2a: Global Alignment
    - Load tree (ete3), embeddings (.pt files), covariance matrix
    - Align embeddings to covariance matrix ordering
    - Raw 960-dim (ESMC) or 1024-dim (ProstT5) embeddings

  Phase 2b: Recursive Mean Shift Testing (recursive_tree_traversal.py)
    - Find candidate splits using tree_alpha dual rule:
      each side >= tree_alpha fraction of species count AND branch length
    - Phylogenetic ANOVA (RRPP) at each candidate node
    - Phylogenetic transformation C^{-1/2} for proper correction
    - Permutation-based p-values (999 permutations)
    - If significant at anova_alpha=0.05, recurse into each clade
    - Output: K stable sub-families with disjoint leaf sets

  Phase 2c: Local pPCA per sub-family
    - PhylogeneticPCA class: R = X^T C^{-1} X / (n-1), eigendecomposition
    - Supports 'cov' and 'corr' modes
    - Respects min_variance and min_components constraints

  Phase 2d: Covariance Shift Test per sub-family (lrt_statistics.py)
    - Find candidate splits within sub-family tree
    - LRT statistic: Lambda = n*log|V_pool| - n_A*log|V_A| - n_B*log|V_B|
    - GLS mean estimation via projection operator P
    - Westfall-Young max-statistic bootstrap (1000 replicates)
    - p_adj = #{max(Lambda*) >= Lambda_obs} / (B+1)

PHASE 3: VISUALIZE
  - Global macro view (sub-family partitioning on tree)
  - Per sub-family: MSA sorted, covariance heatmap, embedding covariance
  - Comparative sequence logos with Q8 secondary structure tracks
  - Structural alignments (predicted + experimental), TM-score matrices

================================================================================
4. KEY ALGORITHMS AND STATISTICS
================================================================================

4.1 Matrix Normal Distribution (MND)
  X ~ MN(M, U, V) where:
  - M = n x p mean matrix
  - U = n x n phylogenetic covariance (from tree, KNOWN)
  - V = p x p coordinate covariance (ESTIMATED)
  - Total covariance = U (x) V (Kronecker product)

4.2 Phylogenetic GLS Mean Estimation
  mu_hat = (1^T U^{-1} 1)^{-1} * 1^T U^{-1} X
  Projection operator: P = U^{-1} - U^{-1}*1*(1^T*U^{-1}*1)^{-1}*1^T*U^{-1}
  Scatter matrix: S = X^T P X

4.3 LRT for V Equality (lrt_statistics.py)
  H0: V_A = V_B = V (same covariance across clades)
  H1: V_A != V_B (different evolutionary dynamics)
  Statistic: Lambda = n*log|V_pool| - n_A*log|V_A| - n_B*log|V_B|
  Where V_hat = S/n (MLE), V_pool = (S_A + S_B)/n

4.4 Westfall-Young Max-Statistic Bootstrap
  For K candidate splits:
  1. Estimate V_pool under global H0
  2. For b = 1..B: simulate X* ~ MN(1*mu^T, U, V_pool)
  3. Compute Lambda for all K splits on each X*
  4. Record max(Lambda*_1, ..., Lambda*_K) per replicate
  5. p_adj = #{max_stat >= Lambda_obs_max} / (B+1)

4.5 Phylogenetic ANOVA (phylogenetic_anova.py)
  - Transform data by C^{-1/2}
  - RRPP (Randomized Residual Permutation Procedure)
  - Trace-based F-statistic: F = (trace(SS_model)/df1) / (trace(SS_resid)/df2)
  - 999 permutations for p-value

4.6 Phylogenetic PCA (PhylogeneticPCA class)
  - Evolutionary VCV: R = X^T C^{-1} X / (n-1)
  - Eigendecomposition of R
  - Supports 'cov' and 'corr' modes
